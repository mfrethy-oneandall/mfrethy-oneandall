# ReGen / ReGeneration — Rancho Campus

## What It Is
ReGeneration is a biblically-based, 12-step recovery/discipleship program. At RC it runs on **Thursday evenings**. The program uses a "Groundwork" (8-week intro) → "Step Study" (9-month) progression.
